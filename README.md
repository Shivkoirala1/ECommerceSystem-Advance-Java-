# 🛒 E-Commerce System — Advanced Programming Coursework

A full-featured desktop e-commerce application built entirely in Java as part of the Advanced Programming module. The system allows customers to browse products, manage a shopping cart, and place orders — while giving administrators full control over the store.

## 🚀 Features
- Product listing with search and filter
- Shopping cart and checkout system
- Order tracking and order history
- Admin dashboard for managing products and orders
- Secure admin login

## 🛠️ Tech Stack
- **Language:** Java 17
- **IDE:** IntelliJ IDEA
- **UI:** Java Swing / JavaFX
- **Database:** MySQL (via JDBC)
- **Build Tool:** Maven
- **Testing:** JUnit 5
- **Version Control:** Git & GitHub
